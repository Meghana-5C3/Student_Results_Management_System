# 🎓 Student Result Management System

A web-based Student Result Management System developed using Django that helps educational institutions efficiently manage student records, subjects, marks, and academic results.

## 🚀 Features

- Student Registration and Management
- Subject Management
- Marks Entry and Result Generation
- Admin Dashboard
- Secure Data Storage using SQLite
- User-Friendly Interface
- Academic Record Tracking

## 🛠️ Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/StudentResultManagement.git
cd StudentResultManagement
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run the Server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
